About

PixelNur is a steganography tool built as a PBL project at KIT's College of Engineering, Kolhapur. It hides AES-256 encrypted messages inside images using LSB substitution — the stego image is visually indistinguishable from the original.

Tech Stack

Component Library Purpose
Image I/O Pillow Read/write images
Pixel ops NumPy LSB array manipulation
Encryption pycryptodome AES-256-GCM + PBKDF2
Web server Flask REST API + file serving
Frontend Tailwind CSS Utility-first styling
Image search Pixabay API Online image search (free key)

Security

CipherAES-256-GCM (authenticated encryption)
Key derivationPBKDF2-HMAC-SHA256, 100,000 iterations
Salt / Nonce16 bytes random per message
Auth tag16-byte GCM tag — detects wrong password or tampering
